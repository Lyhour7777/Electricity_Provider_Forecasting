# Electricity_Provider_Forecasting

## Overview
This project aim to **forecast Electricity for industry production** using Time Series Ananlysis Model like **Auto Regression(AR)**, **Moving Average(MA)**, **ARMA** and **hybrid machine learning** models for efficiency accuracy.

## Installation/Dev

1. Clone Repo
```bash
git clone https://github.com/Lyhour7777/Electricity_Provider_Forecasting.git
cd Electricity_Provider_Forecasting
```

2. Optional: Create virtual environment
```bash
python -m venv .venv --upgrade-deps
source .venv/bin/activate   # Linux / Mac
.venv\Scripts\activate      # Windows
```

3. Install dependencies
```bash
pip install -r requirements.txt
```
